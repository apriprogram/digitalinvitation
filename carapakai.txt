1. Buka terminal di VS Code
2. Jalankan perintah berikut:
npm install

3. Setelah selesai, jalankan:
npm run migrate

4. Lalu jalankan server:
npm start

5. Buka browser dan masuk ke:
http://localhost:3000/admin.html


Semua halaman sudah diupdate menggunakan template "Shalom" dengan animasi modern dan transparan.
